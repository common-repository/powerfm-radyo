== Description ==

Bu eklenti sayesinde Wordpress blogunuzda PowerFM kurulmaktadır.

== Installation ==

1. Wordpress Admin panelinde, EKLENTİLER > Yeni EKLE ve  'PowerFM' olarak arama yapın.

2. 'Yükle' Tıklayın.

3. Wordpress Admin içinde , Görünüm > Bileşenler, içinde  'PowerFM Radyo’ eklentisini istediğiniz bölüme sürükleyip yerleştirin ve kaydedin..
